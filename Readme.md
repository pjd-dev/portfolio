# Jean Darry Paulette Portfolio

Personal portfolio website for Jean Darry Paulette, a FullStack JavaScript TypeScript Developer. Showcases professional work, skills, and contact information with a focus on performance and accessibility.

## Features
- **Internationalization**: Supports English and French languages.
- **Parallax Hero**: Dynamic background image with smooth parallax scrolling.
- **Accessibility**: Reduced-motion safe for users with motion sensitivities.
- **Legal Pages**: Privacy Policy and Terms of Service rendered from Markdown.
- **Responsive Design**: Optimized for all devices.
- **Fast Loading**: Optimized with Next.js and Vercel deployment.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Stitches
- **Icons**: Lucide React
- **Markdown Rendering**: React Markdown with remark-gfm
- **Validation**: Zod
- **Analytics**: Vercel Speed Insights
- **Deployment**: Vercel
